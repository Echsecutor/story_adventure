# Story Adventure Viewer

A web-based player for interactive story adventures. Try it at https://echsecutor.github.io/story_adventure/viewer/.

- Entry point: `index.html`, main logic: `code.js`
- Supports Markdown text, images, videos, save/load progress
- Files never leave your browser

Example stories to play are available in the [stories collection](../stories/README.md).

Part of [Story Adventure Tools](https://github.com/Echsecutor/story_adventure). See there for license, contribution guidelines, and issue tracking.
